# Creative Pipeline

Ideas move through three stages. Only a limited number of projects should be in **Execution** at once.

---

## Stages

### Stage 1: Exploration
New ideas being researched or considered. Not yet committed. Lives in `idea-parking-lot.md` until ready to move.

### Stage 2: Development
Ideas being actively researched, tested, or scoped. Has passed initial evaluation but is not yet in production. Deliverables and timelines being defined.

### Stage 3: Execution
Projects with committed time and resources. Deliverables and deadlines are active. This list should remain short.

## Pipeline Rules

1. New ideas go to `idea-parking-lot.md` first — not directly into execution
2. A project moves to Development only when it has a clear strategic rationale
3. A project moves to Execution only when it has a defined scope, timeline, and resource commitment
4. Maximum 3–4 projects in Execution at any time
5. When in doubt, protect what's already in Execution before adding more
